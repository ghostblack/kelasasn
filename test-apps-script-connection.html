🔌 Test Koneksi Apps Script - Tripay

Tool untuk mengecek koneksi antara frontend dan Google Apps Script

ℹ️ Informasi:
Tool ini akan mengecek apakah URL Google Apps Script Anda sudah benar dan dapat diakses.

📋 Langkah Setup:

  1. Buka Google Apps Script
  2. Buat project baru atau buka project yang sudah ada
  3. Copy code dari file google-apps-script/TripayProxy.gs
  4. Klik Deploy → New deployment → Web app
  5. Set "Execute as": Me dan "Who has access": Anyone
  6. Copy URL deployment dan paste di bawah ini
  7. Jalankan function setupScriptProperties untuk set kredensial Tripay
